### In case of a security vulnurability

</div>

It'll be much appreciated if you communicate this issue, privately, with Lebogang Phoshoko so potential attackers who haven't noted this loophole don't exploit it. Send an email to [dev.phoshoko.ml@gmail.com](mailto:dev.phoshoko.ml@gmail.com) with the following format:

1. What have you found?
2. The possible dangers. (This is so we can determine how to prioritize the issue)
3. Provide simple steps, guide or explanation on how we can exploit the system through this vulnurability.
4. Any further suggestions.
